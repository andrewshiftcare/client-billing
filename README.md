# ShiftCare Per-Client Billing — Subscription Page Mockup

Interactive HTML mockup showing the ShiftCare subscription page with both **staff billing** and **per-client billing** modes.

## How to use

Open `index.html` in any browser — no build step required.

## Features

- Toggle between **Staff billing** and **Per-client billing** modes
- Select plan: Basic, Professional, Premium, Enterprise
- Adjust staff / client count with live order summary
- Switch between monthly and annual billing (20% saving)
- Fully responsive, matches ShiftCare design language
